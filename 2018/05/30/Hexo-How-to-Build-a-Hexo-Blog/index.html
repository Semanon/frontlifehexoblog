[Hexo] How to Build a Hexo Blog?

2018-05-30

在這之前，寫Tech Blog一直是沒能觸及的動作。 開始寫前端Code後，發現好多的技術文章都無法滿足自己的寫作邏輯。 或許是我文字理解能力開始漸漸低下，真令人難過QQ。

本篇開始會提到如何以HEXO這個呆灣人寫的Blog Framwork， 安裝、指令、以及佈署到github的方式。 除了提供其他人可以參閱以外，也是讓我自己的學習腳步留下痕跡。 未來有疑惑可以再回顧一下。

安裝Node.Js

由於我們需要使用到npm套件管理並安裝Hexo，首先請先安裝nodeJs： https://nodejs.org/en/download/

若是Mac OS則可以透過Homebrew套件管理來安裝： brew install node

這樣你就可以在command line使用npm這個指令囉！ 若要確認版本可以使用以下指令： npm -v node -v

安裝git

恩~這個應該對於大部分人來說不成問題，但還是稍微提一下。 git的官方網址: http://git-scm.com/download

Mac OS的使用者可以透過以下指令： brew install git

安裝好後不要忘了設定你的git喔！ git config --global user.name [your name] //Setup your user name git config --global user.email [your mail address] //Setup your e-mail address
